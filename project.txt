Проект - игра с графическим интерфейсом. Играть можно как с ботами, так и с людьми (не по сети).
Игра происходит в 2D, в виде сбоку. Есть карта, на которой танки по очереди стреляют друг в друга,
регулируя угол орудия и силу стрельбы. У игрока есть количество здоровье. Когда оно заканчивается,
то танк самоуничтожается, игрок проигрывает.
Есть ещё несколько разных идей, которые можно внедрить в дальнейшем.

Управление:
стрелки вверх-вниз - увеличение и уменьшение силы выстрела
стрелки влево-вправо - поворот орудия против часовой стрелки и по часовой стрелке
Enter - выстрел
Все клавиши не с цифровой клавиатуры!

Для запуска нужна библиотека pygame
Запуск производится командой python3 game.py
